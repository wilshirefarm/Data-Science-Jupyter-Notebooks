Q4.

I don't think it's a good idea to use the Erdos-Renyi random graph to model a real-world graph because the node degree distribution does not reflect the same node degree distribution of a real-world graph. Real-world graph node degree distributions follows the power-law distribution where there are a small number of nodes with very high degrees and a lot of other nodes with small degrees. The Erdos-Renyi node degree distribution follows that of a poisson distribution.
